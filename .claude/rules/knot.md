# knot

## knot とは
Cosense とデライトに触発された PKM ライクなツール。

## PKM あらため PTW
knot は PKM ではなく、**PTW(Personal Temporary Workspace)** とのコンセプトを掲げている。

Cosense やデライトのような構造化は想定していないし、永続的なメンテナンス（常に未完成とかあらゆるページが変化するなどと表現されることもある）も想定していない。どちらかと言えばメモやスニペットといった言葉が似合うツールであり、構造化や永続メンテの前段を、書くことで支援する。

## ページという情報単位

### ページについて
- 1-ページ 1-ファイルでつくる
- 文法は scbdown を使う。つまり .smb ファイルをつくる
- ファイル名は `(ID).smd` となる

つまり、ページ名が識別子となる Cosense ではなく、ページに ID を与えるデライトの方式に寄せている。これにより、名前を考えずにページを書いていける。

また scbdown なので、Cosense のようなページ間リンクは `[]` で行えるし、箇条書きも 1-space 1-indent で書きやすい。

### ページの先頭行とタイトルについて
ページの先頭行はタイトルとして扱われる。空行の場合、タイトルが無いと解釈する。

### ID について
8桁の`[0-9a-z]`とする。Windows も想定するため大文字小文字は混在させず、小文字だけにする。

## ページ同士のつながり
scbdown であるため、リンク記法によりページとページをつなげることができる。

リンクは `[z5j2t8r4]` の形で書く。しかし、これではページ名がわからないため、人間が適切な説明を付記する形になる。たとえば `詳細はこちら: [z5j2t8r4]` とか `knotについて: [z5j2t8r4]` のように書くだろう。

## ページの種類

### 通常ページ
単にページと書くことが多い。ファイル名が ID となっているページ。

### 特殊ページ
ファイル名に ID を含まないページを指す。knot 側で用途が指定されており、knot のシステムが必要に応じて更新する。ユーザーが編集することもできる。

## 特殊ページについて

### logs.smd
ページを新規するたびに `YYYY-MM-DD HH:mm:SS` のタイムスタンプつきでページ名を prepend する。

例:

```scbdown
 ...
 2026-05-09 11:51:59 [p6t3z5x8]
 2026-05-09 11:51:02 [z5j2t8r4]
```

scbdown のリストとして書いているため、先頭に 1-indent space があることに注意。また新しいエントリを上に書く prepend 方式を取る。

この logs.smb により、ユーザーは時系列的なアクセスが可能となる。もちろん自分で整理を加えても良い。たとえば以下のように月単位で区切ることもできる:

```scbdown
2026-05
 ...
 2026-05-09 11:51:59 [p6t3z5x8]
 2026-05-09 11:51:02 [z5j2t8r4]
```

logs.smd は knot の要であり、これがないと、いつどのページをつくったかのヒントがない。そのため knot では、logs.smd という形で、ページ新規のたびにそのリンクを時系列で記載する仕組みを取っている。

### index.smd
ユーザーが自由に編集できるエントリーポイントまたは机に相当する。

典型的には今現在アクティブなページへのリンクを並べるはずだ。あくまでも temporary workspace の前提で、その俯瞰を支援する机と考えるべきである。したがって、main.smd や desk.smd や desktop.smd といった「そこ自体で作業をしてしまいそうになる名前」は意図的に控えた。むしろ、リンク集としての意味合いが強く、エントリーポイントとしても想起しやすい index.smd を採用した。

## 操作
ページつくる系:

- new: ページを新規
- lew: ページを新規 + リンク元にリンク挿入
    - 今開いているページには、新規したページ xxx のリンク `[xxx]` を挿入する
    - 名前は link + new から
- cutout: 選択範囲を切り出し
    - ページ xxx を lew して、選択部分を切り取りして、xxx に貼り付け
- duplicate: 今開いてるページと同じ内容で new する
- clone: duplicate と同じだけど lew をする

ページを移動する系:

- neopen: `[xxx]` を開く。xxx が存在しない場合は、ファイルをつくらずに開く
    - scbdown の操作
    - 名前は new and open から
